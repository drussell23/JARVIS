Message Intent Types

Agent Log Intents

  • metric — threshold breach / anomaly
  • observation — pattern or state change
  • error — failure report
  • graduation — tool reached count >= 3

Persona Reasoning Intents

  • observe — synthesize specialist logs
  • propose — suggest a solution
  • challenge — raise an objection
  • support — endorse a proposal
  • validate — safety/Iron Gate review
  • consensus — declare agreement
