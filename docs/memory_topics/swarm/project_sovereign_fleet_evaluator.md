# Sovereign Fleet Evaluator (2026-06-19, branch `fleet/sovereign-evaluator`, NOT merged)

**Why built:** Benchmarks proved `Qwen3.5-397B` (the line-67 `DOUBLEWORD_MODEL` cold-start default) is fast, never errors, and emits ZERO valid code (reasons in prose). The existing DW selection layer (discovery + latency-promotion) is BLIND to output correctness — `dw_promotion_ledger` promotes on latency, nothing measures AST-validity. So a fast-useless model is never demoted. Operator authorized a self-updating evaluator (Option 1: Advisory→auto-graduate) instead of hardcoding model names in `brain_selection_policy.yaml`.

**Honest reframe that shaped it:** ~70% of the operator's "Fleet Orchestrator" spec ALREADY ships — `dw_catalog_client.py` discovers `/v1/models` every 30min (403/502-safe, graduated Slice E), the `dw_models:` YAML arrays were PURGED 2026-04-27 (catalog owns facts, YAML owns policy), `dw_promotion_ledger`/`dw_ttft_observer` do latency math with dynamic-N gates. The GENUINE gap was the **correctness axis**. Built additively, REUSING discovery + `graduation_orchestrator.persist_flag_to_env` — did NOT rebuild discovery/ledger or rip out YAML.

**Architecture (4 new leaf modules + 1 binding, all `backend/core/ouroboros/governance/`):**
- `fleet_quality_battery.py` — PURE in-memory AST armor: `CODEGEN_PROMPT`/`CLASSIFY_PROMPT`/`EXPECTED_LABEL="ENRICH"`, `code_quality_pass` (= `is_ast_valid AND NOT has_semantic_placeholder`), `label_adherence`. **NEVER exec/eval/file-write** — `ast.parse` only (adversarial `os.system` string test proves canary never created). Detects `...`/`pass`/`raise NotImplementedError` bodies + docstring-only ≠ placeholder.
- `fleet_calibration_store.py` — `QualityScore` frozen dc (ast_pass_rate, label_adherence, ttft_ms, tok_per_s, sample_count, updated_at), `ewma_update` (alpha 0.4), `valid_tok_per_s`=tok/s×ast, `triage_fitness`=label/ttft×1000. Persist `.jarvis/fleet_calibration.json` atomic. **Cross-kind seed-poisoning fix (load-bearing):** untouched field on brand-new model left UNSEEDED (NaN) so the other kind's first probe seeds directly instead of EWMA-blending vs poisoning 0.0; `_clamp_unseeded` NaN→0.0 on save/score/all_scores so NaN NEVER hits JSON (verified strict-valid). PURE `fleet_rerank` (reorders only scored models, unscored keep slot index, no-op <2 scored), `graduation_ready` (winner beats default by margin OR default ast<0.2), `fleet_apply_rerank` (fail-soft). Daily-spend ledger: `add_spend`/`spend_today` UTC-day rolling.
- `fleet_evaluator.py` — async idle driver. `FleetEvaluator(model_caller, store, idle_check, clock, snapshot_loader, default_model, flag_persister)` all injectable. `calibrate_models` (codegen+classify probe each, in-memory validate, EWMA record, spend-record), `maybe_calibrate(now)` (short-circuits on master-off/not-idle/over-cap BEFORE any probe), `_maybe_graduate` (stable-cycle counter → `persist_flag_to_env("JARVIS_FLEET_EVALUATOR_AUTHORITATIVE","true")`). Live `default_model_caller` (aiohttp late-import → DW `/v1/chat/completions`, fail-soft). Emits `fleet_calibrated`/`fleet_graduated` SSE.
- BINDING: `provider_topology.dw_models_for_route` — all 3 model-returning paths funnel through module-level `_fleet_guarded(route,result)` (gated on `fleet_authoritative_enabled()`, fail-soft pass-through). Empty `return ()` paths untouched. **OFF byte-identical.**
- `flag_registry_seed.py` +13 `JARVIS_FLEET_*` flags; `ide_observability_stream.py` +2 event types (`fleet_calibrated`/`fleet_graduated`).

**Gates:** `JARVIS_FLEET_EVALUATOR_ENABLED` (compute, default false) + `JARVIS_FLEET_EVALUATOR_AUTHORITATIVE` (routing, default false, auto-flipped by graduation). `JARVIS_FLEET_DAILY_USD_CAP` (0.50, enforced via store ledger) + `JARVIS_FLEET_PROBE_USD_PER_MTOK` (0.40 estimate).
